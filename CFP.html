The 12th International Conference on Awareness Science and Technology (iCAST 2023) will be hosted by Chaoyang University of Technology (CYUT), Taichung, Taiwan, in winter 2023. The conference provides a forum for researchers, engineers, and scientists for discussing awareness computing and exchanging ideas, opinions, and the latest results in this emerging new field. For paper submissions and more information, please visit the conference web at (here). Topics of interest include, but are not limited to, the following:

• A : Awareness Science :
o A1 : Awareness mechanism and modeling
o A2 : Machine learning/deep learning
o A3 : Pattern recognition and image understanding
o A4 : Neuro-computing/perception/cognition
o A5 : Fuzzy logic/granular computing/rough set
o A6 : Agent computing/swarm intelligence
o A7 : Ontology/semantics
• B : Awareness Technology :
o B1 : Context/situation/background/semantics awareness
o B2 : Intention/emotion/preference awareness
o B3 : Location/position/goal awareness
o B4 : Power/energy/capability/resource awareness
o B5 : Weakness/risk/danger awareness
o B6 : Safety/security/privacy/trust awareness
o B7 : Chance/opportunity awareness
o B8 : Knowledge/tacit knowledge engineering
• C : Awareness Related Applications :
o C1 : Social network
o C2 : Tele-communication/ubiquitous computing
o C3 : Image/video/speech/signal processing
o C4 : Overlay/P2P networking/M2M
o C5 : Big data/web/text mining
o C6 : E-business/e-government/e-learning
o C7 : Health/child/elderly care
o C8 : Smart grid/transportation/public services
o C9 : Device/automatic control/robotics
o C10 : Cloud computing/internet of things
o C11 : Cognitive radio
• D : The others :
Any other topic about information technology is welcome.
